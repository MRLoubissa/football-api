
# Football Performance API ⚽

A simple REST API built with FastAPI to manage football players and their performances.

## 🚀 Live Demo

API: https://football-api-tylf.onrender.com
Docs: https://football-api-tylf.onrender.com/docs

## 📌 Features

- User authentication (JWT)
- Create, read, update and delete players
- Manage performances
- Protected routes

## 🛠️ Tech Stack

- Python
- FastAPI
- SQLite
- JWT Authentication
- Render (deployment)

## ▶️ Run locally

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
